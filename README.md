# tModLoader Wiki

欢迎来到 tModLoader Wiki 文档仓库！本仓库包含了超过 100 个 Markdown 文档，全面覆盖 tModLoader 的技术指南、使用教程和开发参考。

## 项目概述

tModLoader 是一个为 Terraria 社区开发的模组加载器，允许玩家创建和使用游戏模组来扩展游戏内容。本 Wiki 文档仓库旨在为不同层次的用户提供全面的参考资料：

- **玩家**：学习如何安装、管理和使用模组
- **模组开发者**：掌握模组开发技术，从基础到高级
- **贡献者**：了解如何为 tModLoader 本身做出贡献

## 使用说明

### 浏览文档

本仓库中的每个 `.md` 文件都对应一个独立的 Wiki 页面。您可以：

1. **按类别浏览**：根据上述分类目录找到相关主题
2. **直接搜索**：在 GitHub 仓库中使用 `T` 键或搜索框进行文件搜索
3. **使用链接**：在 Wiki 页面之间通过内部链接导航

### 文档格式规范

- 所有文档使用 Markdown 格式编写
- 标题层级：`#` 用于一级标题，`##` 用于二级标题，以此类推
- 代码示例使用 Triple backticks (```) 并标注语言
- 技术术语在首次出现时会提供解释

### 更新与维护

本文档库由 tModLoader 社区维护，主要更新来源包括：

- tModLoader 版本更新带来的 API 变化
- 社区贡献的新教程和参考文档
- 现有文档的勘误和改进

## 贡献指南

我们欢迎社区成员为 Wiki 文档做出贡献！

### 提交更改

1. **Fork 本仓库**：点击 GitHub 页面右上角的 Fork 按钮
2. **创建分支**：建议使用描述性的分支名，如 `fix-typo-item-class` 或 `add-new-tutorial`
3. **进行更改**：编辑或添加 Markdown 文件
4. **提交 Pull Request**：描述您的更改内容和原因

### 文档编写规范

- 使用清晰、简洁的中文表达
- 技术术语首次出现时使用括号标注英文原文
- 代码示例应包含必要的注释
- 确保代码块的语法高亮正确（如 `csharp`、`json` 等）
- 图片使用相对路径引用

### 注意事项

- 请勿在文档中包含任何个人信息或敏感内容
- 保持文档风格的一致性
- 提交前请检查拼写和语法
