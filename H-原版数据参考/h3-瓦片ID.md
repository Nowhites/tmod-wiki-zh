提醒一下，您可以通过编写 `TileID.Name` 或 just `Number` 来使用这些值，**不要**编写 `TileID.Number`：
```cs
recipe.AddTile(TileID.WorkBenches);     // 正确
recipe.AddTile(18);                     // 正确
recipe.AddTile(TileID.18);              // 错误
```
