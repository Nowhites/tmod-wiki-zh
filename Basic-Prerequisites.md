# 基础先决条件
在查阅任何基础级别指南之前，请阅读以下内容，以便您为基础级别概念做好准备。

## 语法
基础指南不需要太多编程知识，但它们确实需要基本的C#语法理解。请阅读并内化以下内容。
* [C#语法教程](https://www.tutorialspoint.com/csharp/csharp_basic_syntax.htm)

## 文本编辑器
另一件您需要的是文本编辑器。您的操作系统附带了一个，但有更好的可用。以下是更强大的文本编辑器列表。（适用于Windows、Linux和Mac）
* [Notepad++](https://notepad-plus-plus.org/) - 仅限Windows
* [Sublime Text](https://www.sublimetext.com/)
* [Vim](http://www.vim.org/)
* [Visual Studio Code](https://code.visualstudio.com/)
* [GNU Emacs（及其迭代版本）](https://www.gnu.org/software/emacs/)

为什么我们需要好的文本编辑器？好的文本编辑器可以多标签打开多个文件，突出显示语法和关键字，并提供行号。此外，这些编辑器中的大多数（如果不是全部）都带有可扩展设置和键盘快捷键。（有时甚至有插件/附加组件！）所有这些功能将使模组制作变得更加容易。请看下面了解与Notepad相比Notepad++的能力有多强：
> ![](http://i.imgur.com/sViDiZ2.png)

## Visual Studio或其他IDE呢？
您可能听说过Visual Studio。我们稍后将在[中级先决条件](https://github.com/tModLoader/tModLoader/wiki/Intermediate-Prerequisites)中讨论IDE。如果您熟悉编程或充满信心，请随意跳过文本编辑器阶段，直接进入IDE阶段。

## 绘图程序
最后一件您需要的是图片编辑器。我们将在这个程序中制作所有精灵。MS Paint（Windows 11之前）不能工作，因为它无法保存透明像素。请参阅以下模组制作者常用的程序：
* [Krita](https://krita.org/en/) -- Krita是一个免费的开源光栅图形编辑器，能够制作像素艺术精灵，参考[此视频](https://youtu.be/OmnpKQITm3I)
* [GNU图像处理程序(GIMP)](https://www.gimp.org/) -- 免费开源跨平台图像编辑器，支持几乎所有内容，但初始配置复杂。参考[视频演示](https://youtu.be/s8HDwkXq6jk?t=65)
* [Inkscape](https://inkscape.org/) -- 免费开源矢量图形编辑器，参考[此视频](https://www.youtube.com/watch?v=Se7WVuyIEnU)
* [Adobe Photoshop](www.adobe.com/products/photoshop.html) -- 能力很强，但价格昂贵，并非专为精灵工作而设计
* [Piskel](http://www.piskelapp.com/) -- 免费且在精灵制作者中相当受欢迎
* [Aseprite](https://github.com/aseprite/aseprite) -- 开源，意味着它是免费的，但您必须自己编译，因为他们不免费分发副本。您也可以在Steam上购买Aseprite，地址[这里](https://store.steampowered.com/app/431730/Aseprite/)。另外，Aseprite是为精灵制作而构建的，它相当容易和直观地使用。
