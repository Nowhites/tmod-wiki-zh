# 命名 ID 集合

命名 ID 集合促进了模组协作，而无需模组依赖。这被称为临时协作。模组可以使用命名 ID 集合来"标记"其模组中的内容，其他模组可以读取和对这些标记采取行动。

| 内容 ID | 键 | 类型 | 默认值 | 模组用法 |
|----------|----------|------|---------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `ItemID` | ExampleMod/FlamingWeapon | `bool` | `false` | `ExampleMod` - 当武器击中任何东西时随机输出"Hahahah, burn!"。 <br> `OtherMod` - ?? <br> <b>注意：这是教学示例，请不要实际使用它。</b> |
| `ProjectileID` | HolyLight/IndestructibleProj | `bool` | `false` | `HolyLight` - 当移除敌人弹幕时，它不会被移除 |
| `` | | `` | `` | `` -  |
